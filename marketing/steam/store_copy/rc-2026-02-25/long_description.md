# Raven DevOps Fishing

Raven DevOps Fishing is a single-player harbor-to-sea progression game focused on clean controls, readable feedback, and repeatable high-value catch loops.

## Core Loop
- Launch from harbor with your selected ship and hook tier.
- Cast, hook, and reel fish across escalating depth and distance tiers.
- Return to harbor to sell inventory and reinvest earnings into upgrades.

## Progression
- Unlock stronger hooks and ships as catch performance improves.
- Track level, XP, objective progress, and catch history from the profile panel.
- Replay tutorial flows from profile controls when you need a refresher.

## Build and Session Focus
- Deterministic save/load behavior for clean startup and existing profile continuity.
- Keyboard-first menu flows with profile/settings routing and pause-state continuity.
- Non-Steam fallback path for local and CI validation environments.

## Media Alignment
- Store media payload references five gameplay screenshots from Boot, Cinematic, Main Menu, Harbor, and Fishing scenes.
