# Store Assets Checklist

## Required Assets
- Capsule images (main/header/small): required.
- Store screenshots: required.
- Trailer: optional for first publish, recommended.
- Library assets: required.

## Copy Checklist
- Short description and full description are aligned with current feature set.
- Genre/tags and controller support metadata are reviewed.
- Minimum/recommended system requirements are reviewed against hardware baseline.

## Review Gates
- Verify visual style consistency with in-game capture.
- Verify no placeholder-only assets are represented as final marketing deliverables.
- Verify ownership and escalation are documented in `docs/STEAM_RELEASE_COMPLIANCE_CHECKLIST.md`.
