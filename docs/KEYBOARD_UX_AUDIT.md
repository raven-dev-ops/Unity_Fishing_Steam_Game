# Keyboard UX Audit

## Areas
- Main menu
- Settings/Profile/Pause menus
- Harbor interactables
- Fishing controls

## Checklist
- Initial focus is visible on open.
- Arrow/WASD navigation does not get stuck.
- Enter always confirms current focus.
- Esc consistently backs out / pauses.
- Focus aura tracks active item without jitter.

Record blockers with scene, exact key sequence, and expected vs actual behavior.
