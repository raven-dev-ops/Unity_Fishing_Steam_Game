# Steam Age Rating and Content Descriptor Evidence

This folder versions per-RC records for Steam age-rating/content-descriptor review.

## Required Per-RC Artifact
- `release/steam_descriptors/<rc-tag>/descriptor_manifest.json`

## Manifest Minimum Fields
- `rc_tag`
- `reviewed_at_utc`
- `reviewed_by`
- `formal_rating_required`
- `descriptor_matrix` (violence/gore/sexual/nudity/drugs/gambling/ugc)
- `reassessment_triggers`

Current baseline record:
- `release/steam_descriptors/rc-2026-02-25/descriptor_manifest.json`
