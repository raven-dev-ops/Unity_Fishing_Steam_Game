# Release Compliance Package

This directory versions legal/compliance artifacts per RC and launch candidate.

## Package Location
- `release/compliance/<rc-tag>/`

## Required Files
- `compliance_manifest.json`
- `legal_signoff.md`

## Package Expectations
- Rights verification notes for shipped art/audio/fonts.
- Third-party attribution references.
- EULA/privacy disclosure requirement summary.
- Owner + UTC review timestamp.
